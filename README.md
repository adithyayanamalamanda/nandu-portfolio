# Gontla Lakshmi Narasimha - Portfolio

A modern, responsive portfolio website for an Embedded Systems & IoT Developer.

## Features

- **Responsive Design**: Automatically adjusts for mobile, tablet, laptop, and desktop screens
- **Dark Theme**: Professional dark cyberpunk aesthetic
- **Smooth Animations**: Page loader, scroll reveal, and interactive elements
- **Contact Form**: Functional contact form with validation

## Tech Stack

- HTML5
- CSS3 (Custom properties, Grid, Flexbox, Media Queries)
- JavaScript (Vanilla)

## Sections

1. **Hero** - Introduction with name, title, and call-to-action
2. **About** - Personal background and specifications
3. **Skills** - Technical skills and expertise
4. **Projects** - Featured engineering projects
5. **Achievements** - Certifications and recognitions
6. **Leadership** - Community involvement
7. **Education** - Academic background
8. **Contact** - Contact information and form

## Getting Started

Simply open `index.html` in any modern web browser.

## Responsive Breakpoints

- **Mobile**: < 480px
- **Tablet**: 481px - 768px
- **Laptop**: 769px - 1200px
- **Desktop**: > 1200px
